李世优写作

  • 考试安排：

上半场：阅读 + 听力，十分钟休息
下半场：口语 + 写作

  • 托福写作：

综合写作（20分钟，150 - 225词），阅读后听材料，再写作
+
独立写作（30分钟，300词）

  • 机考规范:
  1. 不要空格,齐头写
  2. 段与段之间空一行
  3. 提示词
  4. 人称尽量用第三人称
  5. 标点(与前面单词不能空格,与后面的单词空一格

tpo ： 官方样题（难度和题型差不多

机经：真题集

独立写作：（八种题目类型）

1.黑五类

题目中出现 最高级、always、only、never、ban/stop

破题点：让反反（新的概念要和题目的点做对比

提示词 :

让步段 : admittedly

转折 : however

递进 : furthermore

中立：on the one hand , on the other hand

主体段

分论点 + 分论据（原因、举例、假设、对比） + 小总结

拿分点：

  1. 结构完整（没有小总结不影响分数

  2. 分论点（1.要和题目有明确关系。2.段落内总起。3.是个观点）简洁明了

    写完论点之后对论点进行展开：

    • 创设问题法，解释论点中的疑问(2个左右
    • 分类讨论细化

  3. 论证方法

举例 : 1.已经发生 2.大量细节

假设 : 一个情境

原因 : 和分论点之间形成明确的因果链

分类讨论：一分为二

e.g 。一个城市的old，historic建筑应该保护还是拆毁建新建筑

破题：态度：中立（取决于老房子的类型（有无经济价值

2.横向对比

两个不同主体在同一方面的对比

破题：中立：主体段每一段都要用横向对比

3.纵向对比

同一个主体在不同时间的对比

  • 独立写作评分：

三个档次：1-16，18-25，24-30
想25分+，至少400词+。

  • 题目由三部分组成：
  1. 问题，必答（agree/disagree , which the best , which prefer）
  2. 题目本身外的观点（95%的陷阱来自题目）
  3. 写作指引（大部分都是具体原因例子支持论点：原因 + 例子、假设、对比。和三选一结合可能会要求用自身的例子）例子不要用名人的例子，用身边的
  • 写作分五段：首段 + 三个主体段+ 结尾段

四分钟：

  1. 审题（三部分）

  2. 构思（不带个人倾向，判断题目的合理性和不合理性）写作结构：一边倒，让步-反驳-反驳、反驳-让步-让步，中立

    让步：同意题目观点
    反驳：不同意题目观点
    不同意题目观点：让步-反驳-反驳（让步-转折-递进），让步部分体现辩证的逻辑思维
    同意：反驳-让步-让步

    中立：主体段两段。仅横向对比可以用中立。

  3. 列提纲

1.态度
2.写作结构
3.每个主体段的分论点 + 分论据的简单提示

开头段
  • 作用：给核心论点

背景句子 + 引入题目 + 个人观点

  • 背景句子：

    核心词法：确定一个核心词，描述核心词的重要性。

结尾段
  • 作用 : 总结写作结构 , 核心观点完整表达

提示词 : in conclusion ,to sum up
