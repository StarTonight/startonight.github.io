李世优写作

  • 考试安排：

上半场：阅读 + 听力，十分钟休息
下半场：口语 + 写作

  • 托福写作：

综合写作（20分钟，150 - 225词），阅读后听材料，再写作
+
独立写作（30分钟，300词）

独立写作：（八种题目类型）

三个档次：1-16，18-25，24-30
想25分+，至少400词+。

  • 题目由三部分组成：
  1. 问题，必答（agree/disagree , which the best , which prefer）
  2. 题目本身外的观点（95%的陷阱来自题目）
  3. 写作指引（大部分都是具体原因例子支持论点：原因 + 例子、假设、对比。和三选一结合可能会要求用自身的例子）例子不要用名人的例子，用身边的
  • 写作分五段：首段 + 三个主体段+ 结尾段

四分钟：

  1. 审题（三部分）

  2. 构思（不带个人倾向，判断题目的合理性和不合理性）写作结构：一边倒，让步-反驳-反驳、反驳-让步-让步，中立

    让步：同意题目观点
    反驳：不同意题目观点
    不同意题目观点：让步-反驳-反驳（让步-转折-递进），让步部分体现辩证的逻辑思维
    同意：反驳-让步-让步

    中立：主体段两段。仅横向对比可以用中立。

  3. 列提纲

1.态度
2.写作结构
3.每个主体段的分论点 + 分论据的简单提示

开头段
  • 作用：给核心论点

背景句子 + 引入题目 + 个人观点

  • 背景句子：

    核心词法：确定一个核心词，描述核心词的重要性。

完
