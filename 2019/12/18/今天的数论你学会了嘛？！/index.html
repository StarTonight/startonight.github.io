gcd，lcm

辗转相除法

lcm(S / a, S / b) = S / gcd(a, b)

扩展欧几里得

ax + by = gcd(a,b)必有解，所以ax + by = gcd(a,b) * k必有解

即ax + by = d(d能整除gcd(a,b)，数学符号为gcd(a,b) | d)

已知a,b，求一组x,y满足ax + ay = gcd(a,b)

数论四大定理

威尔逊定理、欧拉定理、中国剩余定理（孙子定理）、费马小定理

威尔逊定理

当且仅当p为质数：( p -1 )! ≡ -1 ( mod p )

即p为质数时能被(p - 1)! + 1整除

欧拉定理

若n,a为正整数，且n,a互质，即gcd(a,n) = 1，则a^φ(n) ≡ 1 (mod n)

φ(n) 是欧拉函数，是求 1到n - 1 中 与n互质的数的数目

中国剩余定理

给出一元线性同余方程：

x mod m1 = a1
x mod m2 = a2
x mod mn = an

中国剩余定理说明：假设整数m1,m2, … ,mn两两互质，则对任意的整数：a1,a2, … ,an，方程组 (S)有解

费马小定理

p是质数，不能整除a，则a(p - 1) mod p = 1，能整除，则a(p - 1) mod p = 0

即，若p是质数，且a,p互质，那么 a的(p-1)次方除以p的余数恒等于1。

费马大定理，断言当整数n > 2时，关于x, y, z的方程 xn + yn = zn 没有正整数解

逆元

逆元类似于倒数。

a,p互质时， a * x mod p = 1，x即a关于p的逆元，用inv(a) 表示。

(a / b) % p = (a * inv(b) ) % p = (a % p * inv(b) % p) % p

逆元求法一：

用费马小定理，a(p - 1) mod p = 1，两边同除a得a(p - 2) mod p = 1 / a，

写作：a(p - 2) mod p = inv(a)

逆元求法二：

拓展欧几里得：a * x + b * y = 1，a，b互质时定有解，x即inv(a)，y即inv(b)

证明：

ax % b + by % b = 1 % b

ax % b = 1 % b

ax = 1 (mod b)

逆元求法三：

当p是个质数的时候有，inv(a) = (p - p / a) * inv(p % a) % p
